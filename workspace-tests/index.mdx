## ðŸ”‘ Key Concepts

### Reactive Data Binding
Components communicate through a reactive data store using `dataKey`:
```mdx
<UserInput dataKey="my_data" agentName="workspace-agent" />
<TextDisplay dataKey="my_data" />
```

### Multi-Agent Orchestration
Chain multiple agents together:
```mdx
<UserInput dataKey="input" agentName="agent-a" />
<DataProcessor dataKey="input" outputKey="processed" agentName="agent-b" />
<TextDisplay dataKey="processed" />
```

### Data Flow Pipeline
```
Input â†’ Agent A â†’ dataKey1 â†’ Processor â†’ Agent B â†’ dataKey2 â†’ Display
```
