# Test 3: JSON Parsing & Display

This test shows how to generate JSON data from AI and parse it for display.

---

## Step 1: Generate JSON

<AgentExecute
  dataKey="raw_json"
  agentName="workspace-agent"
  query="Generate a JSON array of 3 sample products with fields: name, price, category, description. Format as valid JSON in a code block."
/>

---

## Step 2: Raw Response

<TextDisplay
  dataKey="raw_json"
  showLoading={true}
/>

---

## Step 3: Parse JSON

<JsonParser
  dataKey="raw_json"
  outputKey="parsed_products"
  autoExecute={true}
/>

---

## Step 4: Display as JSON

<JsonDisplay
  dataKey="parsed_products"
  showLoading={true}
  fallback="Waiting for parsed data..."
/>

---

## Step 5: Display as Table

<DataTable
  dataKey="parsed_products"
  columns={[
    { key: 'name', label: 'Product Name' },
    { key: 'price', label: 'Price' },
    { key: 'category', label: 'Category' },
    { key: 'description', label: 'Description' }
  ]}
/>

---

## Pipeline Flow

```
AgentExecute → raw_json → JsonParser → parsed_products → JsonDisplay + DataTable
```
