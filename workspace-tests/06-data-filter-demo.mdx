## Filter 1: Developers Only

<DataFilter
  dataKey="users"
  outputKey="developers"
  filter="item => item.role === 'developer'"
  autoExecute={true}
/>

<DataList
  dataKey="developers"
  emptyMessage="No developers found"
/>

---

## Filter 2: High Earners (Salary > $80,000)

<DataFilter
  dataKey="users"
  outputKey="high_earners"
  filter="item => item.salary > 80000"
  autoExecute={true}
/>

<DataTable
  dataKey="high_earners"
/>

---

## Filter 3: Young Managers (Age < 35 AND role = manager)

<DataFilter
  dataKey="users"
  outputKey="young_managers"
  filter="item => item.age < 35 && item.role === 'manager'"
  autoExecute={true}
/>

<DataList
  dataKey="young_managers"
  emptyMessage="No young managers found"
/>

---

## Filter Expressions

DataFilter accepts any JavaScript expression that returns boolean:

- Simple: `item => item.age > 30`
- Complex: `item => item.salary > 50000 && item.role === 'developer'`
- Array methods work too: `item => ['San Francisco', 'New York'].includes(item.city)`
