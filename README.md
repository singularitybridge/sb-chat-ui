# Singularity Bridge - Custom Chatbot Platform

## Overview

This platform is designed to empower companies to create their own AI-powered bots tailored to their specific needs. Built using TypeScript and React, with the speedy Vite as the build tool, this application offers an intuitive interface for creating user accounts and bots with customizable capabilities.

## Features

- **Bot Customization**: Companies can create bots and customize their capabilities. Examples include:
  - Calendar and appointment management
  - Phone conversations using Twilio integration
  - and much more..
- **Bot Personality Definition**: Companies can define the bot's personality, e.g., "you are a helpful secretary," to tailor the interaction style.
- **User Handling**: User accounts are created and handled, allowing for multiple users to interact with bots simultaneously.


## Getting Started

To get the platform up and running on your local machine, follow these steps:

1. Clone the repository:

  ```bash
    git clone https://github.com/aviosipov/sb-chat-ui.git
  ```

2. Navigate to the project directory:

  ```bash
    cd sb-chat-ui-main
  ```

3. Install dependencies:

   ```bash
    npm install
   ```

4. Start the development server:

  ```bash
    npm run dev
  ```
  The application should be running at http://localhost:5173/admin .
